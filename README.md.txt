# 💰 Expense Tracker Application

A **full-stack** expense management application to help users manage personal finances by tracking expenses, setting budgets (overall & category-wise), and viewing spending reports.  

Built with **Spring Boot (Backend)** and **React.js (Frontend)** for a secure, fast, and modern experience.

---

## ✨ Features

### 🔐 User Authentication
- Secure registration & login using **JWT (JSON Web Tokens)**.
- **Profile management** – View & update username, email.
- Change password securely.

### 💵 Expense Tracking
- Add expenses with **name, description, amount, date, and category**.
- View, edit, and delete expenses.
- Search & filter expenses by **name, category, date range**.

### 📂 Category Management
- Create, view, update, and delete categories.

### 📊 Budget Management
- Set **overall monthly budget**.
- Set **category-specific budgets** (e.g., ₹2000 for *Food*).
- **Automatic budget validation** – prevents overspending.

### 📈 Monthly Reports
- Visual breakdown of spending **by category** using pie charts.

### 🎨 Theming
- Light & Dark mode toggle for better user experience.

---

## 🛠️ Technologies Used

### **Backend – Spring Boot**
- Language: **Java 17+**
- Framework: **Spring Boot 3.x**
- REST API: **Spring Web**
- Security: **Spring Security**, JWT, BCrypt
- Database: **MySQL**
- ORM: **Spring Data JPA (Hibernate)**
- Email (explored): **Spring Mail**
- Build Tool: **Maven**
- Utilities: **Lombok**

### **Frontend – React.js**
- Language: **JavaScript (ES6+)**
- State: **React Context API**
- Routing: **React Router DOM**
- HTTP: **Axios**
- Charts: **Recharts**
- Utilities: **jwt-decode**, custom debounce
- Styling: **CSS Modules / Global CSS Variables**
- Build Tool: **Create React App**
